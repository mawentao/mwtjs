对话框

模态对话框，当需要用户处理查看或处理新的事务，而又不希望跳转页面时，可以弹出一个对话框来承载新的页面。



var myDialog = new MWT.Dialog({
    title   : '我的对话框',
    width   : 400,
    top     : 50,
    body    : '
这里是对话框内容
',
    buttons : [
        {label:"确定",handler:function(){alert("确认");myDialog.close();}},
        {label:"关闭",type:'close',cls:'mwt-btn-danger'}
    ]
});
myDialog.open();

弹出动画


new MWT.Dialog({
    title   : '我的对话框',
    width   : 400,
    top     : 50,
    body    : '
这里是对话框内容
',
    animate : 'zoomIn',  /* bounceInDown(默认), zoomIn, flipInX, rubberBand */
    buttons : [
        {label:"确定",handler:function(){alert("确认");myDialog.close();}},
        {label:"关闭",type:'close',cls:'mwt-btn-danger'}
    ]
}).open();

全屏显示的对话框


new MWT.Dialog({
    title      : '我的对话框',
    fullscreen : true,
    body       : '
这里是对话框内容
'
}).open();
