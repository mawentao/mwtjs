消息提示

消息提示向用户显示一些需要关注的信息，包括 通知提示，警告提示，确认提示，进度条提示。

通知提示

通知提示是一种不打断用户当前操作的轻量级提示，通常展现成功、错误反馈等信息。



mwt.notify("1.5秒自动销毁啦",1500);

不同主题的通知


mwt.notify("1.5秒自动销毁啦",1500,"info");
mwt.notify("1.5秒自动销毁啦",1500,"warning");
mwt.notify("1.5秒自动销毁啦",1500,"danger");
mwt.notify("1.5秒自动销毁啦",1500,"success");
mwt.notify("正在加载数据...",1500,"loading");

警告提示

警告提示用于展现需要关注的信息。



mwt.alert('警告提示');

弹框动画


MWT.alert({
    title: '消息标题（自定义）',
    msg: '消息内容',
    animate: 'rubberBand'
});

确认提示

确认提示用于当前操作需要用户进一步的确认时展现。



mwt.confirm("确认删除吗？",function(res){
    if (res) {
        alert("确认");
    } else {
        alert("取消");
    }
});

进度条提示

进度条提示在页面顶部显示进度条来提示当前页面的执行进度。



mwt.showProgress();
mwt.hideProgress();
// 设置进度条颜色
mwt.progress_color='orange';
mwt.showProgress();
