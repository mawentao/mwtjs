数据存储

数据存储负责数据的存储和传输。

加载内存数据


var memoryProxy = new MWT.MemoryProxy({
    data: [
        {id:10001, name:'name_1', gender:'男'},
        {id:10002, name:'name_2', gender:'女'},
        {id:10003, name:'name_3', gender:'男'},
        ......
    ]
});
var store = new MWT.Store({
    proxy: memoryProxy
});
store.on('load',function(){
    console.log(store.root);
});
store.load();
